Tools

In our work on A3 we used a variety of tools to share our work and progress with eachother. For communications we almost predominately relied on Ms Teams for both meetings and team messaging. We also used MsTeams as our shared work drive to which team members would add all their individual sections to a dedicated folder in the ‘files tab’. Following this members would send a supplementary message in the chat to inform the group that a section has been added.
In the MS Teams ‘files’ tab is also where we kept the ‘Final Report’ Document which was a collaborative Microsoft Word Document. We also had numerous backup files of this ‘Final Report’ on MS Teams in the case that the main file became corrupted.
Individually most members used Google Docs to write up contextual sections so that the file would be stored in the cloud for security and accessed across numerous devices for convenience.
